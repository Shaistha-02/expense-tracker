# expense-tracker
The Expense Tracker Web Application is a Java-based web application developed using JSP, Servlets, JDBC, and MySQL.
It allows users to register, log in securely, add daily expenses, and view expense details based on different time periods such as today, weekly, monthly, yearly, and total expenses.

The application follows the MVC (Model-View-Controller) architecture for better code organization and maintainability.
A clean and responsive user interface is designed using HTML and CSS to provide a smooth user experience.

Features: 

User Registration and Login Authentication

Add Daily Expenses (amount, category, description, date)

View Expenses by:

Today

Last 7 Days

Monthly

Yearly

Total Expenses

Dashboard with expense summary

Secure database storage using MySQL
Clean and user-friendly UI

MVC Architecture implementation

Technologies Used: 

Frontend: HTML, CSS

Backend: Java, JSP, Servlets

Database: MySQL

Connectivity: JDBC

Server: Apache Tomcat

Architecture: MVC (Model-View-Controller)
